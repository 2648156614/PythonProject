# 物理题随机生成 MVP 方案（低工作量）

> 目标：在**不做复杂系统**的前提下，快速实现“同题型动态出题”，并尽量避免负数、离谱大数、离谱小数等问题。

## 1. 一句话原则

给定一题后，不去检索题库；而是先抽取该题的“物理骨架（公式/未知量/约束）”，再在合理区间内重采样参数，最后做自动校验与重试。

---

## 2. 从“给你一道题”到“生成新题”的标准流程

1. **识别题型骨架**
   - 核心方程
   - 已知量/未知量
   - 题设假设（如忽略空气阻力）

2. **锁定不变量**
   - 知识点不变（解法不变）
   - 仅替换参数、情境词、数值

3. **参数受限采样**
   - 每个变量设最小值/最大值
   - 在范围内随机，不允许无约束随机

4. **结果校验**
   - 数学合法：有限数、无非法计算
   - 物理合法：非负量不为负，量级不过大不过小
   - 教学友好：小数位可控

5. **失败重试**
   - 不通过则重采样
   - 设置最大重试次数（如 10 次）

6. **输出题目文本**
   - 模板填充变量得到新题

---

## 3. MVP 只需要 3 类配置

每个题型维护一个 JSON/YAML 配置（或数据库字段）即可：

### A. 题型元信息
- `template_id`
- `template_name`
- `problem_text_template`
- `solution_formula`
- `answer_count`

### B. 变量范围配置（核心）
- `variables`: 
  - `name`
  - `min`
  - `max`
  - `step`（可选）
  - `allow_negative`（默认 false）

### C. 答案校验配置
- `answer_constraints`:
  - `min_answer`
  - `max_answer`
  - `max_decimals`
  - `non_negative`（默认 true）

---

## 4. 可直接复用的最小配置示例

```yaml
template_id: KIN-001
template_name: 匀加速直线运动-位移
problem_text_template: "某物体初速度为 __v0__ m/s，加速度为 __a__ m/s²，经过 __t__ s，求位移。"
solution_formula: "v0*t + 0.5*a*t*t"
answer_count: 1

variables:
  - name: v0
    min: 0
    max: 30
    step: 0.5
    allow_negative: false
  - name: a
    min: 0.2
    max: 8
    step: 0.1
    allow_negative: false
  - name: t
    min: 0.5
    max: 20
    step: 0.5
    allow_negative: false

answer_constraints:
  min_answer: 0
  max_answer: 2000
  max_decimals: 2
  non_negative: true
```

---

## 5. 工程上最省事的落地顺序（建议）

1. 先做 3 个高频题型（运动学/牛二/功能）
2. 每题型只维护一份范围配置
3. 统一校验器复用所有题型
4. 统一重试逻辑（如最多 10 次）

这样通常就能覆盖你当前“简单可用、不夸张、不明显错误”的目标。

---

## 6. 质量底线（建议强制）

- 输出必须是有限实数（非 NaN/Inf）
- 非负物理量不得为负（质量、时间、路程、能量等）
- 绝对值超阈值直接丢弃重采样
- 答案统一四舍五入（通常 2 位小数）

---

## 7. 后续可渐进升级（非 MVP 必需）

- 按单位自动缩放参数范围（m、cm、km）
- 难度分层（基础/中等/进阶）
- 题型内“情境改写”提高多样性
- 增加维度一致性检查


## 8. 针对当前系统（已存 `solution_formula`）的最小实现清单

你当前数据库已经保存了解题公式，因此不需要再做“题型识别模型”，只需四步：

1. 给变量加范围（可直接写在 `variables` 字段中）：
   - 普通写法：`v0,a,t`
   - 带范围写法：`v0[0,30],a[0.2,8],t[0.5,20]`
2. 用公式计算答案（沿用 `solution_formula`）
3. 做答案约束校验（有限数、单位推断非负、上下限）
4. 不通过就重采样（最多 N 次）

这四步就能显著减少“负数/离谱值/极端小数”的问题，且工作量非常低。
